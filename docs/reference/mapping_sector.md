# mapping_sector

Mapping Sector to UNHCR 2022 RBM Framework

## Usage

``` r
mapping_sector
```

## Format

A data frame with 83 rows and 3 variables:

- `sector_desc`:

  character COLUMN_DESCRIPTION

- `sector_vocabulary_name`:

  character COLUMN_DESCRIPTION

- `sector_rbm`:

  character COLUMN_DESCRIPTION

- `area_id`:

  character COLUMN_DESCRIPTION
