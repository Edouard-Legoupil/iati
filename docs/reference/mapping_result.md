# mapping_result

Mapping Results to UNHCR 2022 RBM Framework

## Usage

``` r
mapping_result
```

## Format

A data frame with 107 rows and 2 variables:

- `result_title`:

  character result_title

- `sector_rbm`:

  character sector_rbm

- `area_id`:

  character COLUMN_DESCRIPTION
