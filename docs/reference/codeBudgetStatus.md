# codeBudgetStatus

Code to denote if the described budget is binding. This is a Core
codelist.

## Usage

``` r
codeBudgetStatus
```

## Format

A data frame with 2 rows and 6 variables:

- `code`:

  double COLUMN_DESCRIPTION

- `name`:

  character COLUMN_DESCRIPTION

- `description`:

  character COLUMN_DESCRIPTION

- `category`:

  logical COLUMN_DESCRIPTION

- `url`:

  logical COLUMN_DESCRIPTION

- `status`:

  character COLUMN_DESCRIPTION
