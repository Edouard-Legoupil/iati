# mapping_sdg

Mapping UNHCR areas to UN SDG

## Usage

``` r
mapping_sdg
```

## Format

A data frame with 26 rows and 2 variables:

- `area`:

  character COLUMN_DESCRIPTION

- `sdg`:

  character COLUMN_DESCRIPTION
